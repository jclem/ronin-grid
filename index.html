Ronin

A fluid SASS starter kit

Ronin is a fluid, auto-responsive grid framework that helps you slice and dice your content with media queries.

The Grid

Ronin's grid is based on my previous work on Flexgrid, but makes a few key modifications to it.

First, while the Flexgrid is based off of floats, the Ronin grid uses inline-block to control the flow of its column elements. This eliminates the need to group content into rows.

Second, while Flexgrid is built to adapt to any base container width and any number of columns, Ronin assumes an 1140px container with 12 columns and breakpoints at 1024px, 768px, 480px, and 479px. All of these can be changed, but require a little bit of work to do so.

Auto-response

Ronin is (optionally) auto-responsive, meaning that as the browser viewport scales down, the number of columns taken up by an element scales up. If an element takes up 4 columns above 1024px, it takes up 6 columns below 1024px.

This means that Ronin includes a set base container width of 1140px and breakpoints at 1024px, 768px, 480px, and 479px (the last one delineating between iPhone in landscape and portrait orientations).

Usage

In brief:

  1. Add gem 'ronin-grid' to your Gemfile (Rails >= 3.1)
  2. Run the installer: rake ronin-grid:install
  3. Import Ronin into your stylesheets: @import 'ronin-grid';
